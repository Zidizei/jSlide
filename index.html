• About

    jslide is a jQuery plugin to create a simple slideshow of list elements, containing either images or other content. It's currently in an early development stage, but basic features are already integrated.

      Using loop

      This slideshow will automatically slide to the next element every 2000 millisecs (it's the default value of speed). If the user clicks, e.g. the Next-Button, the slideshow slides every 500 millisecs.

      Stop the loop $('div#showcase2').slide().toggleLoop();
